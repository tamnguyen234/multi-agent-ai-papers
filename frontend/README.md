# Frontend Client

The frontend interface for the AI Papers Multi-Agent System, built with **React**, **Vite**, and **TypeScript**.

## Features

- **Dashboard**: High-level overview of daily papers, translation statuses, and audio generation.
- **Paper Browser**: Browse, read summaries, and listen to synthesized Audio Abstracts of AI papers.
- **Trend Map**: Interactive 2D scatter graph (via `react-force-graph-2d`) visualizing paper clusters and topics generated by the Trend Agent.
- **Paper QA Chat**: Interactive chat interface to query the contents of local PDFs via the Ollama-based QA Agent.

## Technologies

- React 19
- Vite
- TypeScript
- TailwindCSS (Styling)
- Axios (HTTP Client)
- React Router DOM
- React Force Graph 2D

## Setup & Run

1. Navigate to this directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file (copy from `.env.example`) and set your backend API URL.
4. Run the development server:
   ```bash
   npm run dev
   ```
The app will be accessible at `http://localhost:5173`.
